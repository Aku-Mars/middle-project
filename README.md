#  Kumpulan Proyek Skala Sedang (Middle-Scale Projects Collection)

![Banner](https://raw.githubusercontent.com/Aku-Mars/gambar/main/bannercps.png)

Selamat datang di repositori ini!

Ini adalah kumpulan proyek-proyek skala menengah yang saya kembangkan untuk tujuan pembelajaran, eksperimen, dan eksplorasi berbagai teknologi.

Setiap folder di dalam repositori ini mewakili satu proyek independen. Anda akan menemukan detail lebih lanjut mengenai setiap proyek (deskripsi, fitur, teknologi, cara setup, dan potensi pengembangan) di dalam file `README.md` yang ada di masing-masing sub-direktori.

## Proyek yang Tersedia:

- **📂 `crime_management/`**: Aplikasi desktop untuk manajemen data kriminal menggunakan Python, Tkinter, dan MySQL.
- **📂 `edge_detection/`**: Kumpulan skrip untuk eksplorasi teknik pemrosesan gambar dengan OpenCV dan Matplotlib.
